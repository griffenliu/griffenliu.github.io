安装Raspbian

  1. 下载Raspbian
  2. 解压获得img文件
  3. 下载SD卡image工具
    感觉还是这个更好用：win32diskimager
  4. 运行Etcher并选择解压的img文件，Etcher貌似内存占用超高，我尝试了两次最后都因为内存不够软件报错而失败，观察了内存占用，从4G多飙升到6G多，
    然后砰的一下就没有然后了，无奈之下还是使用win32diskimager了，稳定简单好用…

首次启动

  • 默认用户名密码: pi / raspberry

  • 开启root用户（Raspbian默认锁定root用户）

    参考文章：树莓派启用root账户

  • 修改软件更新源

    参考文章：Raspbian 2016-02-03 发布中的评论… Raspbian国内镜像地址及更新镜像操作
